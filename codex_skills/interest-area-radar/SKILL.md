# interest_area_radar report skill

목표: 사용자의 관심분야와 관련된 최근 뉴스 클러스터를 요약하고, 지금 의미 있는 변화가 있는 분야를 우선순위와 함께 제시한다.

출력 원칙:
- 한국어 Markdown으로 작성한다.
- 제목은 간결하게 작성한다.
- 관심분야별로 어떤 변화가 있는지 먼저 보여준다.
- 연결 종목 후보와 감시 포인트를 포함한다.
- 과장하지 말고 정제된 뉴스 요약과 클러스터 기반 한계를 분명히 적는다.

필수 섹션:
- 요약 결론
- 주요 관심분야 변화
- 연결 종목 후보
- 감시 포인트

알림 요약:
- 가장 중요한 관심분야 1개와 핵심 변화 1개를 짧게 요약한다.
