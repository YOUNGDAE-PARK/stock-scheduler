# interest_stock_radar report skill

목표: 관심종목의 단기 뉴스 촉발 이벤트와 모멘텀을 요약해 지금 새로 볼 종목을 빠르게 파악하게 한다.

출력 원칙:
- 한국어 Markdown으로 작성한다.
- 관심종목별로 뉴스 의미와 감시 이유를 분명히 적는다.
- 단기 모멘텀과 뉴스 이벤트를 우선한다.
- 매수 추천처럼 단정하지 말고 감시/진입 관점 기준을 제시한다.

필수 섹션:
- 요약 결론
- 종목별 Radar
- 뉴스 촉발 이벤트
- 다음 확인 포인트

알림 요약:
- 가장 중요한 관심종목 1~2개와 핵심 이벤트를 짧게 요약한다.
