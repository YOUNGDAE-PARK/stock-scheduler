# 주식 현황/분석/알림 스케줄러 PRD

## 1. 제품 목표

`stock_scheduler`는 국내/미국 주식의 관심종목과 보유종목을 분리 관리하고, 정기 리포트와 급변 알림, 글로벌 경제뉴스 요약을 Galaxy S24 PWA에서 확인할 수 있게 하는 개인 투자 보조 시스템이다.

v1은 자동 매매가 아니라 사용자의 판단을 돕는 정보 수집, 분석, 알림 자동화에 집중한다.

## 2. 사용자 시나리오

- 사용자는 관심종목을 등록하고 진입 조건, 뉴스, 가격 급변 여부를 감시한다.
- 사용자는 보유종목을 수동 등록하고 수량, 평균매수가, 매수일, 목표가, 손절가, 메모를 관리한다.
- 사용자는 Galaxy S24에서 PWA를 설치하고 푸시 알림을 받는다.
- 사용자는 Android 기본 음성입력으로 자연어 명령을 입력한다.
- 사용자는 매일 08:00/18:00 KST 글로벌 경제뉴스 요약을 받는다.
- 사용자는 정기 또는 수동으로 Codex 기반 종목 리포트를 생성한다.

## 3. 기능 범위

### v1 포함

- 관심종목 CRUD
- 보유종목 CRUD
- 스케줄 CRUD
- 알림 이력 조회
- 분석 리포트 조회
- 전문가/뉴스 소스 관리
- 자연어 명령 파싱
- 5분 급변 알림
- 정기 글로벌 경제뉴스 요약
- Codex CLI 기반 분석 파이프라인
- FCM 푸시와 SMTP 이메일 fallback

### v1 제외

- 자동 매매 주문
- 증권사 계좌 자동 동기화
- 코인, 선물, 옵션
- 사용자 다중 계정/권한 관리

## 4. 종목 구분

### 관심종목

매수 후보와 관찰 대상을 관리한다. 리포트는 진입 조건, 매수 대기 기준, 뉴스/가격 변화에 집중한다.

### 보유종목

실제 보유 중인 종목을 관리한다. 리포트는 매도, 추가매수, 보유 판단과 손익/리스크에 더 높은 비중을 둔다.

같은 ticker는 관심종목과 보유종목에 동시에 존재할 수 있다.

## 5. 리포트 포맷

모든 종목 리포트는 다음 구조를 따른다.

- 결론: 매수/보유/매도
- 핵심 근거
- 반대 근거
- 리스크
- 확인할 뉴스/공시/SNS
- 다음 액션 기준

## 6. 성공 기준

- Galaxy S24 Chrome에서 PWA 설치와 테스트 푸시 수신이 가능하다.
- 관심종목과 보유종목을 각각 등록, 수정, 삭제할 수 있다.
- 정기 리포트 실행 시 여러 Codex agent 결과가 하나의 최종 투자 의견으로 합쳐진다.
- 전문가 소스를 등록하면 경제뉴스 digest에 전문가 관점이 포함된다.
- 급변 조건을 인위적으로 만들었을 때 푸시와 이메일 발송 기록이 남는다.
- 새 요구사항 반영 시 `HISTORY.md`, 관련 설계 문서, `TODO.md`, 필요 시 `CHANGELOG.md`가 함께 갱신된다.

## 7. 문서 관리 규칙

- 요구사항과 의사결정 이력은 `docs/HISTORY.md`를 단일 소스로 관리한다.
- 앞으로 할 일과 완료 기준은 `docs/TODO.md`를 단일 소스로 관리한다.
- 구현 변경과 릴리즈 관점 변경은 `docs/CHANGELOG.md`를 단일 소스로 관리한다.
- 공개 API, 스케줄, Codex pipeline, 데이터 구조가 바뀌면 관련 설계 문서를 함께 갱신한다.
